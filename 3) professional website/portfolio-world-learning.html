WORLD LEARNING

Website to learn geography

I created this website as a game to help people learn geography.

The game is divided into 6 continents (following the 7-continent model, excluding Antarctica).
This division is visible on the homepage and under the "Continents" section in the navigation bar.

By clicking on each continent, users can access the corresponding games page.
Each continent features 4 games: "Guess the capital from the country," "Guess the country from the capital," "Guess the country from the flag," and "Guess the country from the map."

Additionally, Europe and North America include two extra games: respectively, "European Union: true or false?" and a similar quiz for North America.

By clicking on a game title, users are taken to the actual gameplay page, where they can start the game by pressing the Start button.
Each game requires the player to type the correct answer based on the given prompt. For instance, in the "Guess the country from the flag" game, a random flag from the selected continent will be displayed, and the player must enter the corresponding country name.

After submitting an answer and clicking the Check button, the correct answer is shown along with a message indicating whether the playerâ€™s input was correct or not.

At the end of a game session, final statistics are displayed: total score, number of correct answers, percentage of correct responses, and a motivational phrase that changes depending on the score achieved.

The game is available in 3 languages: Italian, English, and Finnish.

There is also a "Study" section featuring cards for each country, including the country's name, capital, map, and flag.

No database is connected to this website, as the goal was to keep it simple, accessible to everyone, and free of data tracking.

Technologies: HTML, CSS, JAVASCRIPT, VUE JS, BOOTSTRAP, DIGITAL OCEAN, NGINX
